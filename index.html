Jekyll boilerplate

Jekyll boilerplate with jekyll-assets (used for JS/CSS compression, cache busting, etc.) and bootstrap.

Setting up the environment

  1. install rvm (if you don't have it yet) with \curl -sSL https://get.rvm.io | bash -s stable --ruby so we don't screw up system ruby (don't forget to source ~/.rvm/scripts/rvm)
  2. git clone
  3. cd jekyll-boilerplate
  4. create rvm project files with rvm --ruby-version use 2.1.1@jekyll-boilerplate --create
  5. install gems with bundler by running bundle (you may need to add gem 'therubyracer' to Gemfile)

Running the site locally

bundle exec jekyll serve --watch

Deploying

Run deploh.sh which uses rsync, but before that you of course have to tweak deploy.sh and _config-remote.yml to meet your needs.
